Ejemplo real — app para Parkinson

Caso práctico · Nersy · Fitness para Parkinson La situación:
Nersy quiere una app de ejercicios para sus pacientes con Parkinson. Con temporizador, instrucciones claras, colores grandes.

Lo que hace Claude Code con GitHub:
1. Busca en GitHub: "exercise timer app" — encuentra 3 opciones gratuitas
2. Descarga la mejor con un comando: git clone [url]
3. Lee todo el proyecto y le explica a Nersy cómo está organizado
4. Cambia colores, agrega ejercicios específicos para Parkinson, adapta los tiempos

Resultado: App personalizada en horas, no semanas. Base de un proyecto probado por miles de personas.
