Sección 1 — Qué Es Claude Code

01

Qué Es Claude Code

Los conceptos básicos antes de tocar nada.

↓ para profundizar
01 · Concepto

Claude Code, En Una Frase

Claude Code es el asistente de IA de Anthropic que trabaja directamente con los archivos y carpetas de tu computadora — en vez de un chat en una página web.

Pensalo como: un asistente sentado en tu teclado que lee, escribe y organiza archivos por vos — pero siempre pide permiso antes de hacer algo.
01 · Concepto

Carpeta de Proyecto, Prompts, Sesiones

  • Carpeta de proyecto — la carpeta donde trabaja Claude Code. Solo toca archivos ahí adentro.
  • Prompt — lo que le escribís, en español simple, pidiendo lo que necesitás.
  • Sesión — una conversación completa, desde que lo abrís hasta que cerrás la consola.
Permisos: antes de editar o crear algo, Claude Code te pregunta. Vos decidís si aprobás.

Sección 3 — Abrir la Consola

Abrir la Consola en Windows

Para quien nunca la abrió — paso a paso, sin dar nada por sabido.

↓ para profundizar
03 · Concepto

¿Qué Es la Consola?

La consola (también llamada "terminal" o "PowerShell") es una ventana donde le escribís instrucciones a la computadora en texto, en vez de hacer clic en íconos.

No hay riesgo de romper nada por abrirla y mirar. Solo pasa algo cuando escribís un comando y presionás Enter.
03 · Abrir

Cómo Abrirla en Windows

1
Hacé clic en el ícono de Windows, abajo a la izquierda
o presioná la tecla Windows
2
Escribí "PowerShell"
Windows te muestra la app mientras escribís
3
Hacé clic en "Windows PowerShell" y presioná Enter
Otra forma: clic derecho sobre el ícono de Windows → elegí "Terminal" o "Windows PowerShell" del menú.
03 · Cómo Se Ve

Sección 4 — Instalar Desde la Consola

04

Instalar Claude Code Desde la Consola

Cómo usar la consola para instalar, y qué está pasando en cada paso.

↓ para profundizar
04 · Cómo Funciona

Cómo Funciona la Instalación

Le pegás un comando a la consola, presionás Enter, y la consola descarga el programa de internet y lo deja listo para usar escribiendo claude desde cualquier carpeta.

"Instalar globalmente" (la -g del comando) significa que después de este paso, el comando claude va a funcionar en cualquier carpeta de tu computadora, no solo en una.
04 · Requisito

Antes de Instalar — Node.js

El método más común necesita Node.js (18 o más nuevo) instalado primero.

node --version
Ves un número (ej. v18.x) → ya lo tenés, seguí a la próxima diapositiva.
Ves "no se reconoce" → descargalo de nodejs.org (versión LTS), instalalo, y cerrá/abrí la consola de nuevo.
04 · Instalar

El Comando de Instalación

1
Pegá este comando en la consola
npm install -g @anthropic-ai/claude-code
2
Presioná Enter y esperá
Van a pasar varias líneas de texto — es normal
3
Cerrá la consola y volvé a abrirla
Así reconoce el nuevo comando
4
Escribí para iniciarlo
claude
¿Prefiere no usar npm? Alternativa de un solo comando, sin Node: irm https://claude.ai/install.ps1 | iex
