Sección 1 — Qué Es Claude Code

01

Qué Es Claude Code

Las palabras y el modelo mental que necesitás antes de que el resto tenga sentido.

↓ para profundizar
01 · Concepto

Claude Code, En Una Frase

Claude Code es el asistente de IA de Anthropic que corre en tu computadora y trabaja directamente con archivos, carpetas y comandos — en vez de una ventana de chat en un sitio web.

Pensalo como: un asistente muy capaz sentado en tu teclado, que puede leer tus archivos, escribir otros nuevos, y ejecutar programas — pero solo después de que le escribas instrucciones y apruebes lo que hace.
No es solo para programadores. La gente lo usa para escribir, organizar archivos, armar documentos, automatizar tareas repetitivas, y sí, también para escribir código — no necesitás saber programar para empezar.
01 · Concepto

Terminal y CLI, Explicados

Terminal

Una ventana simple en blanco y negro donde escribís comandos en vez de hacer clic en íconos. En Mac se llama "Terminal", en Windows es "PowerShell" o "Símbolo del sistema".

CLI

"Command-Line Interface" — el nombre técnico para escribir comandos de texto. Claude Code es una herramienta CLI: le hablás escribiendo en la terminal, no haciendo clic en botones.

No vas a romper nada por abrir una terminal y mirar alrededor. Escribir un comando y presionar Enter es la única forma en que algo sucede.
01 · Concepto

Carpeta de Proyecto, Prompts, Sesiones

  • Carpeta de proyecto — la carpeta de tu computadora donde trabaja Claude Code. Lee y edita archivos solo dentro de esa carpeta (a menos que le indiques lo contrario).
  • Prompt — la instrucción que escribís, en español simple, describiendo qué querés que haga.
  • Sesión — una conversación con Claude Code, desde que lo abrís hasta que cerrás la terminal. Recuerda todo lo que dijiste dentro de esa sesión.
01 · Concepto

Herramientas y Permisos

Claude Code puede leer archivos, editar archivos, y ejecutar comandos en tu computadora — estas acciones se llaman herramientas.

Vos mantenés el control. Antes de editar un archivo o ejecutar algo nuevo, Claude Code pide tu permiso. Podés aprobar una vez, aprobar siempre para ese tipo de acción, o decir que no.

Elegí Cómo Vas a Pagar

OpciónQué esSirve para

02 · Preguntas de Facturación

Preguntas Frecuentes sobre Pagos

  • ¿Necesito tarjeta de crédito? Sí, para Pro, Max, o crédito en Console — ningún plan gratuito cubre el uso de Claude Code.
  • ¿Qué cuenta uso cuando Claude Code me lo pida? El mismo email/contraseña (o cuenta de Google) con el que te registraste en la diapositiva anterior.
  • ¿Puedo cambiar después? Sí — cancelá o cambiá de plan cuando quieras desde la configuración de tu cuenta en claude.ai.
  • ¿Me van a cobrar de más sin avisar? Pro/Max son cuotas mensuales fijas. La facturación de Console es por uso — podés poner un límite de gasto en la configuración de Console si te preocupa.
Sección 3 — Instalar Claude Code

03

Instalar Claude Code

Paso a paso, para Mac, Windows y Linux.

↓ para profundizar
03 · Requisito Previo

Antes de Empezar — Node.js

El método de instalación estándar de Claude Code necesita Node.js (versión 18 o más nueva) ya instalado en tu computadora.

1
Revisá si ya lo tenés
node --version
2
Si ves un número de versión (ej. v18.x o mayor) — pasá a la siguiente diapositiva
3
Si ves "command not found" — descargá Node.js
nodejs.org → descargá la versión LTS → ejecutá el instalador
¿No tenés Node.js? No hay problema. Saltá a la última diapositiva de esta sección para un método de instalación que no lo necesita.
03 · Mac / Linux

Instalar en Mac / Linux

1
Abrí Terminal
Aplicaciones → Utilidades → Terminal (Mac)
2
Ejecutá el comando de instalación
npm install -g @anthropic-ai/claude-code
3
Esperá a que termine
Es normal que pasen varias líneas de texto
4
Iniciarlo
claude
03 · Windows

Instalar en Windows

1
Abrí PowerShell
Menú Inicio → escribí "PowerShell" → Enter
2
Ejecutá el comando de instalación
npm install -g @anthropic-ai/claude-code
3
Cerrá y volvé a abrir PowerShell
Refresca la terminal para que reconozca el nuevo comando
4
Iniciarlo
claude
03 · Alternativa

Alternativa — Instalador Nativo (Sin Node)

Si instalar Node.js te parece demasiado, usá el script instalador propio de Anthropic en su lugar.

# Mac / Linux — pegar en Terminal
curl -fsSL https://claude.ai/install.sh | bash

# Windows — pegar en PowerShell
irm https://claude.ai/install.ps1 | iex
Esto descarga Claude Code directamente — sin Node.js, sin npm, un solo comando, un solo paso.
Sección 4 — Verificar Que Funcionó

04

Verificar Que Funcionó

Tres chequeos rápidos antes de confiar en la instalación.

↓ para profundizar
04 · Chequeo

Revisá la Versión

claude --version
El éxito se ve así: un número de versión impreso, ej. 1.x.x.
El fallo se ve así: "command not found" o "no se reconoce" — mirá la sección de Problemas Comunes a continuación.
04 · Chequeo

Ejecutalo en una Carpeta de Prueba

1
Creá una carpeta nueva, descartable
mkdir claude-test  →  cd claude-test
2
Lanzá Claude Code dentro de ella
claude
3
Se abre una pestaña del navegador — iniciá sesión
Usá la cuenta de la Sección 2
4
Volvé a la terminal — deberías ver un cuadro de prompt
04 · Chequeo

Enviá Tu Primer Prompt

> Creá un archivo llamado hello.txt que diga "Funciona"
Si funciona: Claude Code pide permiso para crear el archivo, lo aprobás, y hello.txt aparece en la carpeta.
Eso es todo — instalación confirmada de punta a punta: cuenta, conexión, permisos, y escritura de archivos funcionando.

Problemas Comunes y Soluciones

Los errores más frecuentes en usuarios sin conocimientos técnicos, y la solución exacta para cada uno.

05 · Errores

Errores de "Comando No Encontrado"

VesQué significaSolución
'npm' is not recognizedNode.js no está instaladoInstalá Node.js desde nodejs.org, reiniciá la terminal
'claude' is not recognizedLa instalación no terminó, o no se reinició la terminalCerrá y volvé a abrir la terminal, probá de nuevo
'node' is not recognizedNode instalado pero el PATH no se actualizóReiniciá la computadora, no solo la terminal
05 · Errores

Errores de Permisos (EACCES)

En Mac/Linux, una instalación global de npm puede fallar con un montón de texto rojo que menciona EACCES — un error de permisos.

Solución más simple: saltá npm por completo y usá el script instalador nativo en su lugar (Sección 3, última diapositiva) — evita este error por completo.
Si querés seguir usando npm: evitá sudo npm install — causa más problemas de los que resuelve. Reconfigurá la carpeta por defecto de npm, o simplemente cambiá al instalador nativo.
05 · Errores

Problemas de PATH y Miedo a la Terminal

  • "Se instaló pero no pasa nada cuando escribo claude" — casi siempre se soluciona cerrando por completo y volviendo a abrir la ventana de la terminal.
  • "La terminal se ve intimidante / puedo romper algo" — no podés dañar tu computadora escribiendo un comando y presionando Enter; en el peor caso, obtenés un mensaje de error.
  • "No sé en qué carpeta estoy" — escribí pwd (Mac/Linux) o cd solo (Windows) para ver tu ubicación actual.
05 · Errores

Login, Antivirus y Confusión de Cuentas

ProblemaSolución
La ventana de login del navegador nunca se abreCopiá la URL que imprime Claude Code y pegala manualmente en tu navegador
El antivirus bloquea el script de instalaciónPermití la descarga temporalmente, o usá el método npm en vez del script curl/irm
Iniciaste sesión pero el uso dice "sin acceso"Iniciaste sesión en una cuenta sin plan Pro/Max activo o sin crédito en Console — revisá la Sección 2

Sección 6 — Consejos y Límites

06

Consejos y Límites

Qué esperar una vez que ya lo tenés funcionando.

↓ para profundizar
06 · Límites

Bueno Saber

  • Claude Code solo toca archivos dentro de la carpeta desde donde lo lanzaste — siempre ejecutalo desde tu carpeta de proyecto real.
  • Los límites de uso se reinician periódicamente en Pro/Max; la facturación de Console descuenta de tu saldo de crédito en su lugar.
  • Cerrar la terminal termina la sesión — volvé a abrir y ejecutá claude de nuevo para continuar, va a recordar el historial del proyecto.
Consejo: empezá cada tarea nueva describiendo el objetivo en una frase simple — Claude Code va a preguntar antes de hacer algo riesgoso.
Consejo: si en algún momento no estás seguro de qué pasó, escribí /help dentro de Claude Code para ver la lista de comandos incorporados.
