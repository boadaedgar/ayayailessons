Section 1 — What Is Claude Code

01

What Is Claude Code

The words and mental model you need before anything else makes sense.

↓ to dive in
01 · Concept

Claude Code, In One Sentence

Claude Code is Anthropic's AI assistant that runs on your computer and works directly with files, folders, and commands — instead of a chat window on a website.

Think of it like: a very capable assistant sitting at your keyboard, who can read your files, write new ones, and run programs — but only after you type instructions and approve what it does.
Not just for coders. People use it to write, organize files, build documents, automate repetitive tasks, and yes, write code — you don't need to know how to program to start.
01 · Concept

Terminal & CLI, Explained

Terminal

A plain black-and-white window where you type commands instead of clicking icons. On Mac it's called "Terminal," on Windows it's "PowerShell" or "Command Prompt."

CLI

"Command-Line Interface" — the fancy name for typing text commands. Claude Code is a CLI tool: you talk to it by typing in the terminal, not by clicking buttons.

You will not break anything by opening a terminal and looking around. Typing a command and pressing Enter is the only way anything happens.
01 · Concept

Project Folder, Prompts, Sessions

  • Project folder — the folder on your computer where Claude Code works. It reads and edits files inside this folder only (unless you tell it otherwise).
  • Prompt — the instruction you type, in plain English, describing what you want done.
  • Session — one conversation with Claude Code, from the moment you open it to the moment you close the terminal. It remembers everything you said within that session.
01 · Concept

Tools & Permissions

Claude Code can read files, edit files, and run commands on your computer — these actions are called tools.

You stay in control. Before it edits a file or runs something new, Claude Code asks for your permission. You can approve once, approve always for that type of action, or say no.

Payment & Billing FAQ

  • Do I need a credit card? Yes, for Pro, Max, or Console credit — no free tier covers Claude Code usage.
  • Which login do I use when Claude Code asks? The same email/password (or Google account) you registered with in the previous slide.
  • Can I switch later? Yes — cancel or change plans anytime from your account settings on claude.ai.
  • Will I be charged unexpectedly? Pro/Max are flat monthly fees. Console billing is usage-based — set a spending limit in Console settings if that worries you.
Section 3 — Install Claude Code

03

Install Claude Code

Step by step, for Mac, Windows, and Linux.

↓ to dive in
03 · Prerequisite

Before You Start — Node.js

Claude Code's standard install method needs Node.js (version 18 or newer) already on your computer.

1
Check if you already have it
node --version
2
If you see a version number (e.g. v18.x or higher) — skip ahead
3
If you see "command not found" — download Node.js
nodejs.org → download the LTS version → run the installer
No Node.js? No problem. Skip to the last slide of this section for an install method that doesn't need it.
03 · Mac / Linux

Install on Mac / Linux

1
Open Terminal
Applications → Utilities → Terminal (Mac)
2
Run the install command
npm install -g @anthropic-ai/claude-code
3
Wait for it to finish
A few lines of text scrolling by is normal
4
Start it
claude
03 · Windows

Install on Windows

1
Open PowerShell
Start menu → type "PowerShell" → Enter
2
Run the install command
npm install -g @anthropic-ai/claude-code
3
Close and reopen PowerShell
Refreshes the terminal so it recognizes the new command
4
Start it
claude
03 · Alternative

Alternative — Native Installer (No Node Needed)

If installing Node.js feels like too much, use Anthropic's own installer script instead.

# Mac / Linux — paste into Terminal
curl -fsSL https://claude.ai/install.sh | bash

# Windows — paste into PowerShell
irm https://claude.ai/install.ps1 | iex
This downloads Claude Code directly — no Node.js, no npm, one command, one paste.

05 · Errors

Permission Errors (EACCES)

On Mac/Linux, a global npm install can fail with a wall of red text mentioning EACCES — a permissions error.

Simplest fix: skip npm entirely and use the native installer script instead (Section 3, last slide) — it avoids this error completely.
If you want to keep using npm: avoid sudo npm install — it causes more problems than it fixes. Reconfigure npm's default directory instead, or just switch to the native installer.
05 · Errors

PATH Problems & Terminal Fear

  • "It installed but nothing happens when I type claude" — almost always fixed by fully closing and reopening the terminal window.
  • "The terminal looks scary / I might break something" — you can't damage your computer by typing a command and pressing Enter; at worst you get an error message.
  • "I don't know what folder I'm in" — type pwd (Mac/Linux) or cd alone (Windows) to see your current location.

Section 6 — Tips & Limits

06

Tips & Limits

What to expect once you're up and running.

↓ to dive in
06 · Limits

Good to Know

  • Claude Code only touches files inside the folder you launched it from — always run it from your actual project folder.
  • Usage limits reset periodically on Pro/Max; Console billing draws down your credit balance instead.
  • Closing the terminal ends the session — reopen and run claude again to continue, it will remember the project's history.
Tip: start every new task by describing the goal in one plain sentence — Claude Code will ask before doing anything risky.
Tip: if you're ever unsure what happened, type /help inside Claude Code for the built-in command list.
