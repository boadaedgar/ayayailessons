Sección 1 — Por Qué Necesitas VPN

01

Por Qué Necesitas VPN

No es un problema de Claude Code — es el mismo bloqueo que afecta a toda cuenta de Anthropic.

↓ para profundizar
01 · Contexto

El Bloqueo Es Por Sanciones, No Por Censura

Anthropic (la empresa detrás de Claude) niega registro y acceso desde IPs venezolanas por las leyes de sanciones/exportación de EE.UU. — Venezuela está en la misma lista que Cuba, Irán, Corea del Norte, Siria y Rusia.

Claude Code usa la misma cuenta y los mismos servidores que Claude en la web — si tu cuenta o tu conexión están bloqueadas ahí, también lo están para Claude Code.
01 · Qué Bloquea Exactamente

Qué Se Bloquea Por IP Venezolana

MomentoQué revisa Anthropic
Registro en claude.aiRechaza el registro si detecta IP venezolana
Inicio de sesiónPuede volver a revisar la IP en cada login, no solo al registrarte
Uso de Claude Code (CLI)Cada comando habla con el mismo servidor — necesita la sesión activa y la IP aceptada
Por eso la VPN tiene que estar activa no solo al registrarte, sino también cada vez que abrís la consola y usás claude.

02

Preparar Cuenta y Pago

Lo que necesitás tener listo antes de instalar Claude Code.

↓ para profundizar
02 · VPN

Elegir una VPN Confiable

OpciónCosto estimadoNota
Surfshark Recomendado~US$2–4/mes (plan de 1–2 años) · ~US$13/mes sin plan largoLa más barata de las tres, conexiones ilimitadas por cuenta
NordVPN~US$3–5/mes (plan largo) · ~US$12/mes sin plan largoMuy estable, buenos servidores en EE.UU./España
ExpressVPN~US$6–10/mes (plan largo) · ~US$13/mes sin plan largoLa más cara, pero la más simple de usar
Windscribe / TunnelBearUS$0 (gratis con límite de datos)Se corta a mitad de mes o de sesión larga
Para usar Claude Code todos los días, una VPN gratis con límite de datos va a cortar la conexión justo quedando la sesión — mejor invertir en una de pago si el uso es constante.
El plan largo (1–2 años) siempre sale más barato por mes que pagar mes a mes — pero significa pagar todo por adelantado, lo cual complica el pago desde Venezuela (ver diapositiva de pago).
02 · Registro

Registrar la Cuenta con VPN Activo

1
Conectá la VPN a un servidor de país soportado
EE.UU., Colombia, España, etc — antes de abrir el navegador
2
Entrá a claude.ai y hacé clic en "Sign up"
3
Si pide número de teléfono, usá uno extranjero
De un contacto fuera de Venezuela, o un servicio de número virtual
4
Verificá tu email y completá el registro
02 · Pago

Pagar el Plan Pro Desde Venezuela

El plan Claude Pro (~US$20/mes) es necesario — el plan gratis no alcanza para Claude Code.

Una tarjeta venezolana casi nunca pasa la validación de pago de Anthropic. Necesitás una tarjeta o cuenta extranjera: tarjeta virtual internacional, cuenta de un familiar/amigo fuera del país, o servicio de tarjeta prepago en dólares aceptado internacionalmente.
Tip: usá la VPN también al momento de pagar — algunos procesadores de pago revisan la IP además de los datos de la tarjeta.
Sección 3 — Instalar y Usar con VPN

03

Instalar y Usar Claude Code con VPN

El mismo proceso de siempre, pero con la VPN encendida en cada paso — Windows.

↓ para profundizar
03 · Instalar

Instalar Claude Code (Windows)

1
Conectá la VPN primero
Antes de abrir PowerShell
2
Abrí PowerShell
Tecla Windows → escribí "PowerShell" → Enter
3
Instalá Claude Code
npm install -g @anthropic-ai/claude-code
4
Iniciá sesión
claude
Con la VPN desconectada, el login se va a quedar cargando o va a fallar — verificá la VPN antes de escribir claude.
03 · Uso Diario

Cada Vez Que Vas a Usarlo

1
Encendé la VPN
Mismo país/servidor que usaste para registrarte, si es posible
2
Abrí PowerShell en tu carpeta de proyecto
3
Escribí claude y trabajá normal
No hace falta reinstalar nada — la VPN es solo la conexión de internet, Claude Code funciona igual una vez que la sesión inició.
Sección 4 — Mantener la Sesión Estable

04

Mantener la Sesión Estable

Evitar cortes e IPs inconsistentes que puedan marcar la cuenta.

↓ para profundizar
04 · Buenas Prácticas

Reglas Simples

  • Usá siempre el mismo país de servidor — cambiar de EE.UU. a España a Colombia en días seguidos puede parecer sospechoso para la cuenta.
  • Elegí un servidor cercano (EE.UU. este, o España) — menos distancia real, menos lentitud al escribir y recibir respuestas.
  • No cambies de servidor a mitad de una tarea larga — mejor terminá la sesión, cambiá, y volvé a iniciar claude.
04 · Si Se Corta

Si la VPN Se Cae a Mitad de Tarea

1
Claude Code va a mostrar un error de conexión
o se va a quedar "pensando" sin responder
2
Reconectá la VPN al mismo país que tenías
3
Cerrá y volvé a abrir Claude Code
claude
El trabajo dentro de la carpeta no se pierde — solo se corta la conexión, no los archivos ya creados.
