MCP = tu archivo Figma como API de herramientas

Figma implementa el Model Context Protocol exponiendo su API de archivos como un conjunto de herramientas (tools) invocables por cualquier agente compatible — Claude Code, Cursor, Windsurf. El agente lee y escribe en Figma igual que vos, pero desde el editor de código.

En términos de dev: es un servidor MCP que wrappea la Figma REST API. Tu agente llama get_design_context(fileKey, nodeId) y recibe el árbol de componentes, variables y estilos — sin scraping, sin capturas.

Flujo real — design-to-code

Agente invoca en secuencia 1 get_design_context(fileKey, nodeId)
→ devuelve colores, tipografía, layout, componentes del frame

2 get_variable_defs(fileKey)
→ devuelve Figma Variables como tokens (primary: #f5a623, spacing-md: 16px)

3 Agente genera JSX/CSS con valores exactos del archivo
→ sin alucinaciones de color, sin valores inventados

Resultado: componente pixel-perfect en el primer intento. Code Connect mapea tu <Button> real al componente de Figma — no uno genérico.
